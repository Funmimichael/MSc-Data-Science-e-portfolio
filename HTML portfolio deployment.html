2. Deploy to GitHub Pages

Here’s the step-by-step:

Step 1 — Create a GitHub Repository

  1. Go to GitHub and log in.

  2. Click New repository.

  3. Name it something like my-portfolio.

  4. Set it to Public.

  5. Click Create repository.

Step 2 — Upload Your Files

You have two main options:

Option A: Drag-and-drop

  • On your repository page, click Add file → Upload files.

  • Drag your index.html (and any images, CSS, etc.) into it.

  • Commit changes.

Option B: Using Git (recommended if you know Git)

bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/my-portfolio.git
git push -u origin main

Step 3 — Enable GitHub Pages

  1. In your repository, go to Settings (gear icon).

  2. Scroll to Pages in the left sidebar.

  3. Under Source, choose Deploy from a branch.

  4. Select main branch and / (root) folder.

  5. Click Save.

Step 4 — Access Your Website

  • After a minute or so, your portfolio will be live at:

    perl
    https://USERNAME.github.io/my-portfolio/
